# Starting a Game

1. Run `./server`.
2. Run `./client`.
3. Enter your player name.

---

# Game Rules

* **One Word:** There is one shared word per round.
* **Turns:** Players take turns guessing one letter at a time.
* **Scoring:** Correct (+1 point, keep turn), Incorrect (pass turn).
* **Ending:** A round ends when the word is fully guessed.
